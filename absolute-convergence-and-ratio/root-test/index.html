Absolute Convergence

Given any series $ \sum a_n $, consider corresponding series $ \sum |a_n| $

If $ \sum |a_n| $ is convergent, then $ \sum a_n $ is called absolutely convergent.

Theorem

If a series $ \sum a_n $ is absolutely convergent, then the series is also convergent.

$$ \sum |a_n| \text{ convergent } \Rightarrow \sum a_n \text{ convergent } $$

Conditionally Convergence

A series $ \sum a_n $ is called conditionally converges if $ \sum a_n $ converges but $ \sum |a_n| $ diverges.

The Ratio Test

Given a series $ \sum a_n $ that

$$ L = \lim_{n \rightarrow \infty} \bigg| \frac{a_{n+1}}{a_n} \bigg| $$

then the series:

  • Converges absolutely if $ L < 1 $
  • Diverges if $ L > 1 $
  • Ratio test inconclusive if $ L = 1 $

The Root Test

Given a series $ \sum a_n $ that

$$ L = \lim_{n \rightarrow \infty} \sqrt[n]{|a_n|} $$

then the series:

  • Converges absolutely if $ L < 1 $
  • Diverges if $ L > 1 $
  • Ratio test inconclusive if $ L = 1 $
